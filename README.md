# PipeUp 2026

PipeUp 2026 is a full-stack web application.

## Structure

- `/api`: The backend application, built with Laravel as a RESTful API. It uses MySQL and includes features like Laravel Telescope for debugging.
- `/front`: The frontend application, built with Vue 3, TypeScript, and PrimeVue.

## Getting Started

This project uses a Dockerized environment for development. Please refer to the specific README files in the `/api` and `/front` directories for detailed instructions on how to set up, run, and develop each part of the application.
